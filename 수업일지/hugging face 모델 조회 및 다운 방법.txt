--- 허깅페이스 토큰 인증이 필요 없는 경우 ---
(예시1)
self.sentiment_analyzer = pipeline(
            "sentiment-analysis",
            model="snunlp/KR-FinBert-SC",
            tokenizer="snunlp/KR-FinBert-SC"
        )
(예시2)
classifier = pipeline("sentiment-analysis")  // 파이프라인에서 기본적으로 제공하는 모델 및 토큰나이저 다운이 됨

--- 허깅페이스 토큰 인증이 필요한 경우 ---
- vscode 터미널에서 pip install huggingface_hub 설치
- 허깅페이스 사이트 우측 상단 허깅페이스 아이콘에서 settings> Access Token에서 토큰 생성
- 생성한 토큰 복사 
- 터미널에서 huggingface-cli login 입력 후 복사한 토큰을 붙여넣고 엔터
- 성공 메시지 확인 (Login successful)
- 허브의 비공개 모델이나 데이터셋에 접근하거나 허깅페이스 API를 사용 가능
