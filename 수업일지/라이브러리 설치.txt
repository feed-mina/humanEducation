**TensorFlow 2.0 이상을 Windows에서 사용하기 위해서는 Microsoft Visual C++ 재배포 패키지가 필요

Visual C++ 재배포 패키지 다운로드:

Microsoft 공식 사이트 (https://aka.ms/vs/17/release/vc_redist.x64.exe)에서 Visual C++ 재배포 패키지 2015-2022 x64 버전을 다운로드

//설치 과정:

다운로드한 파일(vc_redist.x64.exe)을 실행합니다
"동의함" 체크 후 설치 버튼을 클릭합니다
설치가 완료될 때까지 기다립니다

//설치 확인:
Windows 설정 > 앱 > 설치된 앱에서 "Microsoft Visual C++ 2015-2022 Redistributable (x64)"가 있는지 확인합니다
