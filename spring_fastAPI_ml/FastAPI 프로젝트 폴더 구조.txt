*** 각 파일 및 폴더의 역할 ***
<main.py>
- 프로젝트의 진입점(entry point)
- FastAPI 애플리케이션 초기화 및 CORS 설정
- 라우터 및 미들웨어 설정
- /predict 엔드포인트 정의

<app/dataset/watertot.csv>
- 유입수량 및 전력량 데이터가 포함된 훈련 데이터셋
- 모델 예측에 사용되는 히스토리컬 데이터

<app/model/lstm_model.h5>
- 훈련된 LSTM 모델 파일
- 전력량 예측을 위한 딥러닝 모델

<app/routers/predict.py>
- 예측 관련 라우팅 로직
- API 엔드포인트 정의

<app/services/waterPredict02.py>
- 핵심 비즈니스 로직 구현
- 데이터 전처리, 모델 예측, DB 저장 등의 기능
- predict_water_electricity(): 7일 예측 수행
- get_forecast_data(): DB에서 예측 데이터 조회
- mysql+pymysql://root:비밀번호@localhost:3306/스키마명?charset=utf8 에서 관련 스키마 접속 비밀번호로 변경
