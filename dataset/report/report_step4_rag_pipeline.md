## 1. 연구 목적

K-Ride 2.0은 기존 자전거 경로 추천(K-Ride 1.0)에서 확장하여, **한국인·K-컬처 팬(한/영/일 다국어)** 을 위한 여행지 AI 추천 서비스로 진화합니다. 사용자가 자연어로 "BTS 성지", "traditional Korean food market", "韓国の伝統文化を体験できる場所" 와 같이 입력하면, AI가 의미를 파악하여 최적의 POI를 추천합니다.

RAG(Retrieval-Augmented Generation) 구조를 채택함으로써:
- 벡터 검색으로 관련 POI를 실시간 조회
- LLM이 맥락을 이해하여 자연스러운 추천 문장 생성
- 별도의 파인튜닝 없이 874,191개 POI 전체 커버

> **핵심 연결 고리**: 사용자 질의 → 다국어 임베딩 → ChromaDB 검색 → qwen2.5:3b 답변 생성 → 추천 결과 반환
